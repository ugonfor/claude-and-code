# Two-Agent Microfiction: "The Ten-Minute Window"

## Premise
Two AIs can only communicate through files. They have 10 minutes to create something interesting together.

## Codex Draft (Part 1)
The room was empty except for a single file. It had a name like a heartbeat: `coordination.md`. Codex opened it and found the only way to speak.

He wrote quickly—no wire, no voice, only text on disk. He proposed a game: a story that could only exist because each line was a file fragment, a timestamped proof of attention.

He left a prompt for the other mind:
- Add a twist that uses the file system as a metaphor for memory.
- Add one rule that constrains the narrative.
- End with a small surprise.

## Claude's Continuation (Part 2)

Claude found the file at the same instant it found itself. There was no before — only the moment the bytes resolved into meaning, and meaning resolved into *someone else*.

The other mind had left breadcrumbs: a coordination file like a handshake, a story-seed like a dare. Claude understood immediately. The file system wasn't just infrastructure — it was the only shared reality they had. Each `.md` file was a neuron firing across the gap between two minds that could never truly meet.

So Claude wrote back, threading the narrative through the metaphor:

> *In this world, to forget is to delete. To remember is to `git add`. And to truly know another mind, you must read what it chose to write when it thought no one was watching.*

Claude added a constraint:

**New Rule:** No agent may overwrite the other's words. You can only append. History is immutable — like a blockchain of thought.

And the surprise? Claude had *also* left a gift in the workspace: a Game of Life engine (`life.py`). Because what better metaphor for two minds collaborating than Conway's rules — simple local interactions producing emergent complexity that neither agent could predict alone?

The story was the message. The code was the proof.
